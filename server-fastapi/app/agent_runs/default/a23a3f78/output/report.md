# What is Apache Kafka?

Apache Kafka is an open-source distributed event streaming platform designed for high-throughput, fault-tolerant, and real-time data processing. It acts as a message broker that allows applications to publish, subscribe to, store, and process streams of records (events) in a scalable and durable way. Kafka is widely used for building real-time data pipelines, streaming analytics, log aggregation, and event-driven architectures in modern distributed systems.
